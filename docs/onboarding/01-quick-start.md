# 5 分钟快速体验

本文档让你在 5 分钟内体验 Facio Superpowers 的核心功能。

## 什么是 Facio Superpowers？

一句话：**让 AI 按照团队规范帮你写代码、管需求、做协作。**

传统 AI 编程工具（如 Copilot）只是"补全代码"，而 Superpowers：
- 有完整的开发流程（设计 → 计划 → 实现 → 验证）
- 自动记录技术决策
- 与团队协作工具集成
- 确保代码质量（测试驱动、代码评审）

## 快速体验

### 步骤 1：进入项目目录

```bash
cd 你的项目目录
```

### 步骤 2：启动 Claude Code

```bash
claude
```

你会看到类似这样的界面：

```
╭─────────────────────────────────────╮
│  Claude Code                        │
│  Type your request...               │
╰─────────────────────────────────────╯
```

### 步骤 3：体验 AI 对话

直接用自然语言告诉 AI 你想做什么：

```
我想添加一个用户登录功能
```

**注意观察**：AI 不会直接写代码，而是：

1. **提醒你准备上下文**
   ```
   ⚠️ Development Preparation Required
   Before starting development, I recommend:
   1. Call /prepare-context to find relevant documentation
   ```

2. **询问需求细节**
   - 登录方式是什么？（用户名密码/手机验证码/第三方登录）
   - 需要记住登录状态吗？
   - 登录失败如何处理？

3. **提出设计方案**
   - 展示 2-3 种实现方式
   - 分析各方案优缺点
   - 推荐最适合的方案

4. **获得你的确认后才开始实现**

这就是 Superpowers 的核心：**先想清楚，再动手**。

### 步骤 4：体验核心 Skill

试试输入以下命令感受不同的 Skill：

| 命令 | 效果 |
|-----|------|
| `/prepare-context` | 自动搜索项目中相关的设计文档和决策记录 |
| `/brainstorming` | 进入设计讨论模式，AI 会逐步引导你完善想法 |
| `/verification-before-completion` | 检查代码质量、运行测试、生成文档 |

### 步骤 5：体验 Facio Flow（团队协作）

如果你的项目已配置 Facio Flow，试试说：

```
我有个想法，想给 APP 加个收藏夹功能
```

AI 会自动询问是否创建讨论上下文（Context），这样：
- 想法被记录下来，不会丢失
- 团队其他人可以看到和参与讨论
- 讨论结果可以转化为开发任务

## 关键体验点

通过这 5 分钟，你应该感受到：

| 传统 AI 工具 | Facio Superpowers |
|------------|-------------------|
| 你说什么它就做什么 | 先问清楚再做 |
| 只管写代码 | 设计 → 计划 → 实现 → 验证 |
| 代码写完就结束 | 自动生成文档、记录决策 |
| 每次对话独立 | 讨论内容持久化，团队可见 |

## 常用快捷键

在 Claude Code 中：

| 快捷键 | 作用 |
|-------|------|
| `Ctrl + C` | 中断当前操作 |
| `Ctrl + L` | 清屏 |
| `/help` | 查看帮助 |
| `/clear` | 清除对话历史 |
