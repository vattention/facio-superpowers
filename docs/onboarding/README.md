# Facio Superpowers 新人培训

欢迎加入团队！本培训将帮助你快速掌握 Facio Superpowers —— 我们的 AI 辅助开发框架。

## 你是谁？选择你的学习路径

| 我的背景 | 建议路径 |
|---------|---------|
| 完全新手（不熟悉 Git/命令行） | 从 [00-prerequisites](./00-prerequisites.md) 开始 |
| 有开发经验，没用过 AI 编程工具 | 从 [01-quick-start](./01-quick-start.md) 开始 |
| 用过 Cursor/Copilot，想学 Superpowers | 看 [02-core-concepts](./02-core-concepts.md) 了解差异 |
| 产品经理 | 看 01 → 02 → [产品经理指南](./roles/product-manager.md) |
| 测试人员 | 看 01 → 02 → [测试人员指南](./roles/tester.md) |
