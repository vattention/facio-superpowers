# 产品经理指南

本指南帮助产品经理了解如何使用 Facio Superpowers 进行需求管理、版本规划和进度跟踪。

## 产品经理的核心工作

在 Facio Superpowers 中，产品经理主要负责：

1. **提出和讨论需求**（Context 阶段）
2. **版本规划**（组织 Context 到版本）
3. **进度跟踪**（查看 Task 状态）
4. **验收功能**（确认是否符合验收标准）

---

## 一、需求讨论

### 如何提出新需求？

直接用自然语言描述你的想法：

```
我想给 APP 加一个收藏夹功能，用户可以收藏感兴趣的内容
```

AI 会询问是否创建 Context，确认后进入讨论模式。

### 讨论过程中会发生什么？

AI 会扮演"产品经理助手"的角色，帮你：

**澄清需求**：
- "收藏的内容类型是什么？文章、商品、还是都可以？"
- "用户能看到自己收藏了多少内容吗？"
- "收藏有上限吗？"

**识别边界条件**：
- "如果内容被删除了，收藏列表怎么显示？"
- "未登录用户能收藏吗？"

**提出验收标准**：
```
验收标准：
- [ ] 用户可以在内容详情页点击收藏按钮
- [ ] 收藏成功后按钮状态变为"已收藏"
- [ ] 用户可以在"我的收藏"页面查看所有收藏
- [ ] 用户可以取消收藏
- [ ] 收藏列表按收藏时间倒序排列
```

### 如何确认需求？

当你对讨论结果满意时：

```
验收标准确认了，就按这个做
```

AI 会记录决策，Context 进入"已决策"状态。

### 需求变更怎么办？

如果发现已完成的需求需要修改：

```
之前的收藏功能要改一下，需要支持分组
```

AI 会创建 followup Context，关联原讨论，保持完整追溯链。

## 二、版本规划

### 创建新版本

```
创建 video-editor 的 v0.5 版本，目标是完成视频基础剪辑功能
```

AI 会：
1. 创建版本目录结构
2. 自动列出可规划的需求（已决策但未规划的 Context）
3. 询问要加入哪些需求

### 将需求加入版本

```
把"收藏夹功能"加到 v0.5 版本
```

### 从版本移除需求

```
把"分享功能"从 v0.5 移出，放到后面版本
```

### 查看版本状态

```
video-editor v0.5 的进度怎么样了？
```

AI 会展示：
- 需求完成情况
- 任务状态统计
- 预计完成情况

## 三、进度跟踪

### 查看所有进行中的任务

```
看看现在有哪些进行中的任务
```

### 查看某个需求的任务

```
收藏夹功能的开发进度怎么样了？
```

### 查看某人的任务

```
小王现在在做什么任务？
```

## 四、功能验收

### 验收流程

当开发告诉你"功能做完了"，进行验收：

```
验收一下收藏夹功能
```

AI 会：
1. 显示验收标准清单
2. 引导你逐项确认
3. 记录验收结果

### 验收通过

```
验收通过
```

Task 状态更新为"已完成"。

### 验收不通过

```
有问题：取消收藏后按钮状态没有变化
```

AI 会记录问题，通知开发修复。

## 六、最佳实践

### 需求描述要清晰

**不好的描述**：
```
加个分享功能
```

**好的描述**：
```
用户浏览内容时，希望能分享给朋友。主要场景是分享到微信和复制链接。
分享时要带上内容摘要和缩略图。
```

### 验收标准要具体

**不好的标准**：
```
- [ ] 分享功能好用
```

**好的标准**：
```
- [ ] 详情页右上角有分享按钮
- [ ] 点击分享按钮弹出分享面板
- [ ] 分享面板包含：微信好友、微信朋友圈、复制链接
- [ ] 分享到微信时，卡片显示标题+摘要+缩略图
- [ ] 复制链接后显示"已复制"提示
```

### 及时确认和反馈

- 讨论中 AI 问的问题要及时回答
- 需求变更及时同步，不要私下沟通后忘记记录
- 验收发现问题及时反馈，说清楚具体问题

## 七、常见问题

### Q: 我只是随便问问，不想创建 Context

```
先不创建 Context，我只是想了解一下
```

### Q: 讨论一半要处理其他事情

直接离开就好，下次说"继续讨论 xxx"即可。

### Q: 想看看所有需求的整体情况

```
列出所有 open 状态的 Context
```

### Q: 需求太大，想拆分

```
把这个需求拆分成几个小的
```

AI 会帮你将大 Context 拆分成多个子 Context。
